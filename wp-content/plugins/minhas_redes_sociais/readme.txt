Divulgue suas redes sociais no seu site, com ícones e links.
 
== Description ==

Este plugin ativa um widget com campos personalizados, para você inserir as seguintes redes sociais: 
1. Facebook
2. Twitter
3. Youtube
4. Instagram
 
== Installation ==
 
 Após você baixar o plugin, deve seguir o processo abaixo:
 1 - Ativar o plugin
 2 - Ir na opção widget
 3 - Inserir a URL das suas redes sociais
